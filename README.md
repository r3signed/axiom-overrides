# Axiom Overrides

![image Icon](/src/main/resources/assets/axiom-overrides/icon.png)


---

This mod only works for Minecraft 1.20.1 and *requires* [Axiom](https://axiom.moulberry.com) to be installed.

Contains a collection of overrides for [Axiom](https://axiom.moulberry.com) designed
specifically to avoid conflicts while playing on the [ArdaCraft](https://www.ardacraft.me/)
server.

---

### Overrides
A list of the overrides included in this mod:

- Remove the `Colour Picker` button from the creative inventory
- Remove the `Gradient Helper` button from the creative inventory
